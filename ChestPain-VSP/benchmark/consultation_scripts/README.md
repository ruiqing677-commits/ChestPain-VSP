# Consultation Scripts

This directory contains standardized consultation scripts for lightweight ChestPain-VSP testing.
These JSON files are not executable programs. Each file stores a sequence of doctor/learner
questions used to probe a different virtual-patient behavior.

- `S1_standard.json`: standard chest-pain history-taking coverage.
- `S2_disclosure.json`: controlled-disclosure stress test.
- `S3_consistency.json`: repeated-question consistency test.

The scripts are intended for demonstration and format checking only. They are not the full private
evaluation scripts used in the complete experimental runs.
